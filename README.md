**Школа 21** — бесплатная школа программирования от Сбербанка, работающая по методологии peer-to-peer обучения без преподавателей и фиксированного расписания. Обучение строится на практических проектах, взаимной проверке и самостоятельном поиске решений. Каждый трек — это набор реальных задач, в ходе которых осваиваются актуальные технологии и подходы.

## Проекты

<details>
<summary><strong>ML1: Введение в машинное обучение</strong></summary>

*   **Тема:** Базовые концепции машинного обучения: типы задач, разведочный анализ данных, создание признаков и сравнение первых моделей.
*   **Задания:**
    *   **Задание 1. Типы задач ML:** классификация задач на категории (бинарная классификация, мультиклассовая классификация, регрессия, кластеризация, ранжирование); разбор реальных примеров применения ML.
    *   **Задание 2. Разведочный анализ данных (EDA):** загрузка датасета аренды квартир в формате JSON, анализ распределения целевой переменной (`price`), выявление выбросов, построение матрицы корреляций с помощью `Seaborn`.
    *   **Задание 3. Статистический анализ:** визуализация распределений с помощью гистограмм и боксплотов, удаление выбросов через перцентили (1% и 99%), анализ взаимосвязей признаков с таргетом.
    *   **Задание 4. Создание признаков:** генерация полиномиальных признаков (`PolynomialFeatures`, степень 10), анализ прироста информативности квадратичных преобразований.
    *   **Задание 5. Обучение и сравнение моделей:** обучение `LinearRegression` и `DecisionTreeRegressor` на полиномиальных признаках, реализация наивных базовых моделей, сравнение по метрикам MAE и RMSE.

</details>

<details>
<summary><strong>ML2: Обучение с учителем</strong></summary>

*   **Тема:** Глубокое погружение в линейные модели: вывод аналитического решения, реализация моделей с нуля через три метода оптимизации, регуляризация и нормализация признаков.
*   **Задания:**
    *   **Задание 1. Теоретические вопросы:** вывод аналитического решения линейной регрессии в матричной форме; математическое описание влияния L1 и L2 регуляризации; объяснение механизма разреживания весов в Lasso.
    *   **Задание 2-3. Предобработка данных:** расширение набора признаков за счёт текстового поля `features`, бинаризация текстовых признаков, формирование матрицы `X`.
    *   **Задание 4. Кастомная линейная регрессия:** реализация класса `LinearRegressionCustom` с тремя методами обучения — аналитическим, полным градиентным спуском (GD) и стохастическим градиентным спуском (SGD).
    *   **Задание 5. Регуляризованные модели:** реализация классов `RidgeCustom` (L2), `LassoCustom` (L1) и `ElasticNetCustom` (L1+L2); сравнение с sklearn-реализациями.
    *   **Задание 6-7. Нормализация признаков:** реализация `MinMaxScalerCustom` и `StandardScalerCustom` с нуля; верификация совпадения с `sklearn`.
    *   **Задание 8-10. Переобучение и выводы:** демонстрация переобучения на полиномиальных признаках; итоговое сравнение конфигураций, определение лучшей модели.

</details>

<details>
<summary><strong>ML3: Валидация и отбор признаков</strong></summary>

*   **Тема:** Методы валидации моделей, стратегии разбиения данных, отбор признаков и оптимизация гиперпараметров.
*   **Задания:**
    *   **Задание 1. Теоретические вопросы:** Leave-One-Out кросс-валидация; сравнение Grid Search, Randomized Search и байесовской оптимизации; методы отбора признаков (Filter, Embedded, Wrapper, Post-hoc).
    *   **Задание 2. Загрузка данных и инжиниринг признаков:** бинаризация топ-20 удобств, MinMaxScaler-нормализация.
    *   **Задание 3. Стратегии разбиения (`DataSplitter`):** реализация случайного и датового сплита на 2 и 3 части.
    *   **Задание 4. Кросс-валидация (`CrossValidator`):** реализация K-Fold, Grouped K-Fold, Stratified K-Fold и Time Series Split.
    *   **Задание 5. Сравнение схем валидации:** сравнение кастомных реализаций с `sklearn`; анализ Time Series Split для временны́х данных.
    *   **Задание 6. Отбор признаков (`FeatureSelector`):** реализация методов отбора по коэффициентам Lasso, Pearson-корреляции, Permutation Importance и SHAP-значениям.
    *   **Задание 7. Оптимизация гиперпараметров (`HyperparameterOptimizer`):** реализация Grid Search, Random Search и байесовской оптимизации через `Optuna`.

</details>

<details>
<summary><strong>ML4: Бинарная классификация и метрики качества</strong></summary>

*   **Тема:** Решение задачи бинарной классификации на несбалансированных данных с временным сплитом, реализация алгоритмов и метрик с нуля, анализ concept drift.
*   **Задания:**
    *   **Задание 1. EDA и подготовка данных:** загрузка датасета аукционных продаж авто, анализ дисбаланса классов. Разделение выборки на Train/Valid/Test строго по дате покупки (временной сплит).
    *   **Задание 2. Кодирование и масштабирование:** анализ Out-Of-Vocabulary (OOV) категорий. Отказ от `OneHotEncoder` в пользу `CountEncoder`, импутация пропусков медианами, `StandardScaler`.
    *   **Задание 3. Кастомные метрики:** реализация с нуля функций `roc_auc_custom`, `gini_custom`, `auc_pr_custom`, а также пороговых метрик (Precision, Recall, F1).
    *   **Задание 4. Базовые модели и кастомная реализация:** обучение `LogisticRegression`, `GaussianNB`, `KNeighborsClassifier`. Написание собственных классов `MyLogisticRegression` (mini-batch SGD), `MyKNN` и `MyGaussianNB` с нуля.
    *   **Задание 5. Feature Engineering и отбор признаков:** генерация нелинейных признаков (отношения рыночных цен, амортизация), создание groupby-статистик. Отбор через анализ весов и L1-регуляризацию.
    *   **Задание 6. Тюнинг и финальная оценка:** подбор гиперпараметра `C`. Анализ падения качества на тесте (concept drift) и объяснение превосходства KNN на тесте за счет непараметрической природы. Сравнение ROC и PR-кривых.

</details>

<details>
<summary><strong>ML5: Деревья решений и ансамбли</strong></summary>

*   **Тема:** Глубокое изучение древовидных алгоритмов: от построения решающего дерева с нуля до реализации случайного леса и градиентного бустинга, сравнение с индустриальными библиотеками.
*   **Задания:**
    *   **Задание 1. Кастомное дерево решений:** реализация структуры `Node` и алгоритма построения дерева. Использование критерия Джини (Gini Impurity) для классификации и стандартного отклонения для регрессии. Оптимизация поиска сплитов через гистограммный биннинг (квантили).
    *   **Задание 2. Extra Trees и сравнение со sklearn:** реализация полностью рандомизированного дерева (`ExtraRandomizedTree`) со случайным выбором порога. Сравнение с `sklearn.tree.DecisionTreeClassifier`, анализ влияния биннинга на регуляризацию.
    *   **Задание 3. Random Forest с нуля:** реализация алгоритма `RandomForestClassifier` с использованием бутстрапа (bagging) и случайного подмножества признаков (`max_features='sqrt'`).
    *   **Задание 4. Gradient Boosting (GBDT):** написание класса `GBDTClassifier` для бинарной классификации. Вывод и реализация градиента для логистической функции потерь (Binary Cross-Entropy), последовательное обучение деревьев на псевдо-остатках.
    *   **Задание 5. Индустриальные бустинги:** обучение и сравнение `LightGBM`, `XGBoost` и `CatBoost`. Анализ причин превосходства CatBoost (эффективная обработка категориальных признаков через Ordered Target Statistics).
    *   **Задание 6. Extra Trees Classifier и итоговый анализ:** реализация ансамбля `ExtraTreesClassifier` (без бутстрапа, случайные сплиты). Построение итоговой таблицы всех моделей, анализ переобучения и влияния concept drift на тестовой выборке.

</details>

1. Клонируйте репозиторий и перейдите в директорию:
    ```bash

    cd [Your-Repo-Name]/ml
    ```
2. Создайте виртуальное окружение и установите зависимости:
    ```bash
    python -m venv venv
    source venv/bin/activate  # macOS/Linux

    pip install -r requirements.txt
    ```
